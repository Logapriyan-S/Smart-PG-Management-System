Smart PG Management System

A web-based application to manage Paying Guest (PG) accommodations efficiently. This system helps PG owners manage rooms, tenants, rent payments, and complaints in a simple and organized way.

Features

User registration and login (JWT authentication)

Admin and Tenant roles

Room management (add, update, delete)

Tenant management and room allocation

Rent payment tracking (paid / pending)

Complaint raising and resolution
